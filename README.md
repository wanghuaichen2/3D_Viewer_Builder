# 📁 3D模型查看/生成器（支持PLY/STL/OBJ）

基于Three.js的离线3D模型查看器/生成web。可以同时拖入最多5个PLY/STL/OBJ模型文件，点击按钮即可生成一个**独立的HTML文件**，该文件包含完整的**模型数据**和**3D查看器**，支持按文件名控制模型的显示/隐藏，方便**分享**或离线**查看**。

## ✨ 功能特点

- **多格式支持**：PLY、STL、OBJ（二进制或ASCII）
- **批量处理**：最多同时处理5个模型文件
- **实时统计**：加载PLY/STL/OBJ文件时自动解析顶点数和面数
- **一键生成**：生成一个包含所有模型数据的`view.html`文件，完全离线可用
- **模型控制**：生成的查看器左侧提供复选框，可独立控制每个模型的显示/隐藏
- **友好交互**：左键旋转、右键平移、滚轮缩放，操作直观

>右键平移操作与浏览器外观行为冲突问题：
>关闭edge浏览器行为和功能：浏览器设置->外观->其他外观设置->浏览器行为和功能->启动鼠标手势（取消）

## 🚀 快速使用

1. **打开工具**  
   在浏览器中打开`main.html`。
2. **拖拽或点击选择模型文件**  
   支持`.ply`、`.stl`、`.obj`格式，最多可选5个文件。  
4. **点击“生成 view.html”**  
   等待进度条完成，浏览器会自动下载一个名为`view.html`的文件。
5. **打开生成的查看器**  
   用浏览器打开`view.html`，即可看到所有模型，左侧复选框可控制显隐，右侧调试面板显示加载日志。  

---

## 📂 支持的文件格式

| 格式 | 说明 |
|------|------|
| **PLY** | 支持ASCII和二进制，自动解析顶点/面数 |
| **STL** | 支持ASCII和二进制，自动解析顶点/面数 |
| **OBJ** | 支持ASCII，自动解析顶点/面数 |

> ⚠️ 注意：生成器会将模型文件内嵌到`view.html`中，因此如果模型文件很大，生成的HTML也会很大，建议单个文件不超过50MB。

## 🧩 生成的查看器功能

- **模型控制**：左侧面板列出所有模型文件名，勾选/取消即可显示/隐藏对应模型。
- **调试信息**：右侧面板显示加载过程中的日志，包括模型总数、解码状态、错误信息等。
- **自适应相机**：所有模型加载完成后自动调整相机位置，使模型居中显示。
- **交互方式**：
  - 左键：旋转
  - 右键：平移
  - 滚轮：缩放
- **半透明材质**：每个模型被赋予随机颜色的半透明材质，方便观察重叠部分。

## ⚙️ 技术细节

- **前端**：原生HTML/CSS/JavaScript，无任何构建步骤。
- **3D引擎**：Three.js (r128) + 官方加载器（PLYLoader、STLLoader、OBJLoader、OrbitControls）。
- **数据嵌入**：模型文件数据通过Base64编码内嵌到生成的HTML中，确保单文件离线可用。
